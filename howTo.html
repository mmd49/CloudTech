How Create a docker repository

Step 1 Getting docker

To begin making your repository, you will first need to create an account on Docker. This process can be quickly be done by going to their website(https://www.docker.com) and sign up for free.

After signing up, download Docker from the website. Docker is free and can work on Windows and Mac.

Step 2 Using the Docker Hub

Go back online to your Docker Hub. There on the top right corner, you should see a button labeled Create Repository. Click on it.

You will then be asked whether to make your repository public or private. For a more safe repository, you should click on private. You can also add a description to differentiate from different repositories.

Step 3 Uploading an Image

To upload an image to your new repository, you will have to open the Docker desktop app. Create a pull request for the image that you would like to use. You can do this by going into your command line window and typing “docker pull image.”

To check and see if your image is there, you can use the “docker images” command to see all Docker images on your computer.

Step 4 Making Changes with Apache

To modify your image on docker, you will first need to download apache on docker. Go to the command prompt and run the image to start the Docker container. Then type in “docker run -it ubuntu:latest /bin/bash”. Install all the latest applications in the image container. Then exit out of the container.

Step 5 Using a Docker Website

In your Docker container run “/etc/init.d/apache2 restart” to start the apache program. Then use your preferred browser to access the container and display your website.
